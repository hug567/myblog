### 2、安装卸载

```c
sudo apt install <package>                               //安装软件
dpkg -l                                                  //查看所有已安装软件
dpkg -l | grep -i <name>                                 //查看指定相关软件
sudo apt --purge remove <package>                        //删除软件及其配置文件
apt-cache search <package>                               //搜索软件
```
